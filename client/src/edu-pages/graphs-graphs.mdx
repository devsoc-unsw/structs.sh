## What is a Graph? ğŸŒ

A **graph** is a data structure that consists of a set of nodes (or vertices) and a set of edges connecting these nodes. It is a powerful tool used in computer science to model relationships and networks.

## Basic Terminologies of Graphs ğŸ› 
- **Vertex**: A node in the graph.
- **Edge**: A connection between two vertices.
- **Directed Graph**: A graph where the edges have a direction associated with them.
- **Undirected Graph**: A graph where the edges do not have a direction.
- **Weighted Graph**: Each edge in the graph has a weight or cost associated with it.
- **Adjacency List**: A common way to represent a graph where each vertex stores a list of adjacent vertices.

## Types of Graphs ğŸ“Š
- **Simple Graph**: A graph without loops and parallel edges.
- **Multi-graph**: A graph that may have multiple edges between the same vertices.
- **Complete Graph**: A graph in which every pair of vertices is connected by an edge.

## Advantages of Using Graphs ğŸ‰
- **Modeling Relationships**: Graphs are excellent for modeling complex relationships.
- **Path Finding**: Used in GPS and networking to find the shortest path between two points.
- **Resource Allocation**: Helps in resource management and allocation problems.
- **Network Analysis**: Used for analyzing social, biological, and communication networks.

- The `addVertex` function adds a new vertex to the graph.

Graphs are used in:
- **Social Networks**: To represent and analyze social connections.
- **Google Maps**: For finding the shortest path between locations.
- **Web Crawlers**: To navigate and index the web.
- **Resource Scheduling**: In operations research and logistics.
